# 시스템 프로그래밍 (System Programming)

[![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/99MotiveU/System-programing-ubuntu/blob/main/LICENSE)

## 📘 프로젝트 개요

이 저장소는 **'시스템 프로그래밍 리눅스&유닉스'** (이종원 지음, 한빛아카데미) 교재를 기반으로, **이형원 교수님**과 함께 학습하고 실습한 내용을 정리한 공간입니다.

**리눅스/유닉스 환경에서의 시스템 프로그래밍** 개념을 이해하고, 각 챕터별 핵심 이론과 관련된 실습 코드를 포함하고 있습니다.

## 📚 교재 정보

* **도서명:** 시스템 프로그래밍 리눅스&유닉스
* **저자:** 이종원
* **출판사:** 한빛아카데미

## 📂 저장소 구조

저장소의 각 디렉토리는 교재의 해당 **챕터**에서 학습한 내용과 실습 작업물을 담고 있습니다.

| 디렉토리 | 설명 | 비고 |
| :--- | :--- | :--- |
| **`ch02`** | 파일 및 디렉토리 | |
| **`ch03`** | 파일 입출력 (File I/O) | |
| **`ch04`** | 프로세스 (Process) | |
| **`ch05`** | 프로세스 제어 | |
| **`ch06`** | 시그널 (Signal) | |
| **`ch08`** | 스레드 (Thread) | |
| **`ch09`** | 뮤텍스 및 세마포어 | |
| **`ch10`** | 파이프 및 FIFO | |
| **`ch11`** | 시스템 V IPC: 메시지 큐 | |
| **`ch12`** | 시스템 V IPC: 공유 메모리 및 세마포어 | |
| **`ch13`** | 소켓 프로그래밍 (TCP/UDP) | ch13과 ch14 내용 통합 |

### 📝 참고 사항

* **`ch07`**과 **`ch15`**는 시험 주간 또는 별도의 실습이 없었기 때문에 디렉토리가 생성되지 않았습니다.
* **`ch13`** 디렉토리는 교재의 **13장 및 14장(소켓 프로그래밍)** 내용을 모두 포함하고 있습니다. (교재와 챕터 번호가 다를 수 있습니다.)
